Plajabook consente di inserire all'interno del sito web una comoda interfaccia per la prenotazione degli ombrelloni e dei servizi degli stabilimenti balneari.

== Description ==

PlajaBook è un prodotto pensato per tutti i proprietari di Stabilimenti Balneari che vogliono offrire ai propri clienti la possibilità di prenotare comodamente da casa il proprio ombrellone.
I bagnanti non dovranno fare altro che visitare il tuo sito, scegliere il posto ed effettuare la prenotazione direttamente on-line. Avranno la possibilità di concludere il pagamento in contanti oppure on-line tramite PayPal.

Principali funzionalità:

*   "Mappa" rappresentazione visiva degli ombrelloni con possibilità di configurarla a piacimento
*   "Pagamenti" modalità di pagamenti in contanti o con carta di credito/PayPal
*   "Prenotazioni" consulta lo storico delle prenotazioni ricevute e l'elenco dei clienti
*   "Servizi" consente l'acquisto di servizi aggiuntivi durante la prenotazione
*   "Setup" installazione semplicissima e compatibile con qualsiasi sito web

   
Per utilizzare il plugin è necessario disporre di un account Plajabbok.

== Installation ==

1. Caricare `plajabook.zip` nella cartella `/wp-content/plugins/`
1. Attrivare il plugin attraverso il menù 'Plugins' di WordPress
1. Utilizzare lo short code `[show_plajabook]` all'interno di una qualsiasi pagina
